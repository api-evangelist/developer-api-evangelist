API Evangelist - Stack APIs

This is the developer portal for API Evangelist. I tend to create separate projects, hosted on Github for all my API projects, but here you will find any APIs I've created under The API Stack brand here.

The API Stack APIs all do one thing, and do it well and each API I design is open sourced as a Github project, with supporting documentation, API definition, etc. After I launch each API I will register here under my central developer hub.


These API is maintained by Kin Lane (@kinlane), the API Evangelist. If you have any questions, comments or there are APIs you'd like to see deployed, you can submit them on this projects, Github repo issue management. I use 3Scale API Infrastructure to manage access to my APIs, across all parts of the API Evangelist network.
