This is the API platform, for accessing API Evangelist resources. As with all of my research, my APIs are a work in progress, meant primarily for my own use. However, I encourage you to play with as well, and if you require a specific level of service, please let me know, so I can help make sure and meet your needs.

Here are the steps to get started:

  • Register for a developer account
  • Visit the documentation for the API you wish to integrate with
  • Import postman collections into your postman if you need to play further
  • Submit a Github issue for this repository, or each specific API if you need help

I am continuing to work on the APis, and improve the design, error codes, and underlying data models. I am also working in real-time on the supporting building blocks like documentation, roadmap, etc.

If there is anything I can make smoother, or left out please let me know.
