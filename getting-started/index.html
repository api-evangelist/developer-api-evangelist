This is my personal API platform, that I use to manage API Evangelist. It isn't 100% ready for everyone to use, so I'm not inviting public access to everything just yet. However since part of my goal with this API, is to help showcase some of the best practices, I am making sure it contains all of the essential building blocks that I prescribe to the rest of the space.

If you are really eager to play with it, and are willing to deal with bugs and changes while I am stabilizing, I'm happy to get you some API keys, so you can play. Just submit an issue requsting access.

Here are the steps to get started:

  1. Register for a developer account (not ready yet)
  2. Visit the documentation for the API you wish to integrate with
  3. Visit the code page, and download the language library you need
  4. Import postman collections into your postman if you need to play further
  5. Submit a Github issue for the master repository, or each specific API if you need help

I am continuing to work on the APis, and improve the design, error codes, and underlying data models. I am also working in real-time on the supporting building blocks like documentation, roadmap, etc.

If there is anything I can make smoother, or left out please let me know.
